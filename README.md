# 个人健康管理多Agent助手
基于 Python 实现的轻量级多 Agent 健康管理系统，实现从数据采集、健康分析、方案优化到报告生成的全流程闭环。

## 项目介绍
本项目通过多个智能 Agent 协同工作，帮助用户完成个人健康与减脂管理，自动分析身体数据、饮食、运动与用药情况，并生成个性化优化方案。

## 核心功能
1. 数据采集 Agent：采集身高体重、饮食、运动、用药等多维度数据
2. 健康分析 Agent：计算基础代谢、热量缺口、营养结构、风险预警
3. 方案优化 Agent：生成饮食、运动、用药个性化调整方案
4. 健康报告 Agent：输出周期健康总结与预期效果

## 运行方式
python health_agent_log.py

## 适用场景
- 个人减脂与健康管理
- 多维度健康数据自动分析
- 智能方案生成与提醒

## 项目亮点
- 多 Agent 协同工作
- 全流程自动化闭环
- 轻量化、可直接运行
